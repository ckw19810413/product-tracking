# Product Tracking — Digital Sales Analytics & Revenue Reporting

> A complete digital product sales tracking and weekly revenue reporting system. Monitor your Gumroad sales, analyze trends, and generate revenue reports automatically.

## Overview

This system tracks sales across your digital product portfolio and generates weekly revenue reports. Built for creators and entrepreneurs who sell digital products through platforms like Gumroad.

## Directory Structure

```
product-tracking/
├── data/            # Sales data storage (JSON, CSV)
├── reports/         # Generated weekly revenue reports
├── scripts/         # Automation & data processing scripts
│   └── landing-page/# Next.js landing page for reports
└── research/        # Market research & analysis docs
```

## Getting Started

```bash
# Install dependencies
npm install

# Start the dev server
npm run dev
```

## Features

- 📊 **Sales Dashboard** — Visual breakdown of revenue by product, period, and channel
- 📈 **Weekly Reports** — Auto-generated revenue summaries with trend analysis
- 🔍 **Product Analytics** — Track individual product performance over time
- 💰 **Revenue Tracking** — Monitor MRR, one-time sales, and total earnings
- 📉 **Bottleneck Detection** — Identify conversion drop-offs and growth blockers
