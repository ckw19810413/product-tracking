# Gumroad 上架前最後檢查清單

> 在點擊 Publish 之前，確保所有項目正確無誤。

---

## 📋 產品頁面檢查

### 標題
- [ ] 標題清晰明瞭，不超過 60 字
- [ ] 包含關鍵字（方便 Gumroad 搜尋）
- [ ] 沒有打字錯誤

### 描述
- [ ] 開頭 2 行是 Hook（最重要）
- [ ] 使用 bullet points 列出重點
- [ ] 包含社會證明（如有）
- [ ] 有明確的 CTA

### 圖片
- [ ] 封面圖 1200×628 像素
- [ ] 色彩清晰，文字易讀
- [ ] 手機上看也不模糊

### 價格
- [ ] 價格正確（USD）
- [ ] 有免費預覽
- [ ] 標籤正確

### 檔案
- [ ] 上傳的檔案能正常下載
- [ ] 格式正確（PDF/ZIP）
- [ ] 檔案大小適中

---

## 🔗 連結檢查

- [ ] 複製 Gumroad 產品連結
- [ ] 貼到 `data/gumroad-urls.txt`
- [ ] 確認連結能正常開啟

## 📈 上架後即時動作

1. **分享連結** → 貼到 Twitter/X 個人簡介
2. **確認 Landing Page 連結** → 更新 CTA 按鈕
3. **檢查 Gumroad Dashboard** → 確認頁面公開
4. **測試購買流程** → 用 $1 測試付款（確認金流正常）
